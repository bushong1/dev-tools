# Complete Build and Deployment for OZP

Clones, compiles, configures, and launches:

    * OZP REST API
    * Center
    * HUD
    * WebTop
    * IWC Bus
    

## Helpful Hints:
    * nginx error logs:  `/var/log/nginx/error.log`
    * restart nginx: `nginx -s reload`
    * stop a running grails app: `grails stop-app`
